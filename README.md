# vitepress blog

一个基于 Vitepress 框架的个人简洁博客项目。

<img src="./docs/public/index.png" style="border:1px solid #efefef" />

## 项目特点

- 自定义首页
- 网站加载页
- 全文搜索
- 全文图片放大
- 网站访问量统计
- GitHub评论系统
- 自动配置侧边栏
- 自动打包部署GitHub Pages
- 自动统计文章字数/阅读时间/最近更新时间
- 未完待续......

## 使用方法

```bash
pnpm docs:dev
```

```bash
pnpm docs:build
```
