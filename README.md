# vitepress blog

vitepress 搭建的简约风博客

预览地址：[https://dddhl.cn](https://dddhl.cn)

搭建过程：[blog搭建过程](https://dddhl.cn/2024/blog-change.html)

## 特点

- 自定义首页
- 网站加载页
- 全文搜索
- 全文图片放大
- 网站访问量统计
- GitHub评论系统
- 自动配置侧边栏
- 自动打包部署GitHub Pages
- 自动统计文章字数/阅读时间/最近更新时间
- 未完待续......

## 使用方法

```bash
pnpm docs:dev
```

```bash
pnpm docs:build
```
